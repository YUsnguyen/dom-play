What Does This Page Do?

This webpage is a working application that highlights lines within a script that is specific to the actor. The page is programmed to only highlight an actor's designated lines through the usage of data attributes within HTML. This allows for the 'span' element to be hooked onto JavaScript's querySelector method, which we can then use in as an object in a for loop. I then utilized an event listener to link to the mySpan object to highlight the lines when clicked. Give it a try, and click on one of the lines!
